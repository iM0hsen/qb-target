# qb-target

## تغيير شكل التارقت

تقدر تغيّر شكل التارقت من خلال الخيار الموجود في ملف:  
`init.lua`  
وتحديدًا في **السطر 52**.

شكل التارقت الموضّح في هذا الفيديو:  
https://youtu.be/H_uD_l1GBIM
